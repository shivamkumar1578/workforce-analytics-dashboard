Workforce Analytics Dashboard 📊

🔍 Project Overview
This Power BI project focuses on analyzing workforce attrition and employee risk levels to support data-driven HR decisions.

🎯 Objectives
- Analyze employee attrition trends
- Identify high-risk employee segments
- Understand attrition by department and years at company

🛠 Tools & Technologies
- Power BI
- Microsoft Excel
- DAX

📊 Dashboard Highlights
- Total Employees & Attrition Rate KPIs
- Attrition Count by Department
- Attrition Trend by Years at Company
- Risk Level Distribution (Low / Medium / High)
- Interactive slicers for deep analysis

📂 Repository Structure
- Dataset: HR employee data
- Dashboard: Power BI (.pbix) file
- Screenshots: Dashboard preview
